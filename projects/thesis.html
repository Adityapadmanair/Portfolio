About

A thesis project focusing on interactive visualization of large-scale 2D datamaps directly in a web browser. Designed for researchers and engineers handling scientific data. Project was later bought and published as a software by IMTEK Freiburg for NETCDF data handling in contact mechanics.

How it Works

  • Data collected from NetCDF files using Node.js and Axios.
  • Visualization rendered as color maps on HTML5 canvas with WebGL.
  • As we zoom further in more and more data gets called and loads.
  • It works on the principle that we don't really need the entire data when the user is completely zoomed out.
  • This makes reduce the loading time and makes the transition smoother.
  • Supports seamless zooming, panning, and interaction even for images up to 10GB.
  • Elegant UI built with Tailwind CSS and vanilla JavaScript.
